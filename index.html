01 — E-commerce
SabiFix Artisan Marketplace
UX Research Figma Prototyping React Native Usability Testing
Duration
4 months

Problem Statement

Finding reliable artisans for home services is often difficult, as people rely on informal recommendations that lack transparency in pricing, quality, and availability. At the same time, many skilled artisans struggle with visibility and consistent job opportunities, creating a need for a trusted digital platform that connects customers with verified service providers.

My Role

UI/UX Designer working alongside other Product Designers and a 3-person engineering team. We owned research, information architecture, interaction design, and design system implementation.

Design Process

Conducted user interviews with homeowners and artisans and analyzed existing service marketplace platforms to identify usability gaps. Translated insights into user personas and journey maps, then created low-fidelity wireframes and prototypes. Iterated through usability testing before developing the final high-fidelity UI design

Key Decisions

Simplified the artisan booking process from multiple steps into a quick 4-step flow for faster service requests. Introduced a clear job status tracker, improved service selection and pricing transparency, and added detailed artisan profiles to help users compare skills, ratings, and availability before booking.

02 — PropTech
Haven Homes Website
Product Design Data Viz Design System UX Research Tailwind
Duration
6 months

Problem Statement

Many potential home buyers struggle to find a reliable and user-friendly platform to explore property listings. Existing real estate websites are often cluttered, slow, or lack clear property information, making the search process overwhelming and inefficient for users.

My Role

End-to-end Product Designer and Front-End Developer. I conducted user research, designed the full UI system in Figma, built interactive prototypes, and implemented the responsive interface using React and Tailwind CSS. My focus was creating a seamless browsing experience that allows users to easily discover properties and connect with agents.

Design Process

Conducted research with potential home buyers and renters to understand how they search for properties online. Identified key user needs such as clear property images, transparent pricing, and easy navigation. Designed intuitive search and filtering experiences and built a scalable design system to maintain visual consistency across pages.

Technical Implementation

Built the platform using React and Tailwind CSS to ensure a responsive and modern interface. Implemented optimized image loading for property galleries and created reusable UI components for listings, property cards, and agent contact forms. The result is a fast, accessible, and user-friendly real estate browsing experience.
